# Claim Form Project

This is a responsive React.js single-page application that replicates a legal claim submission form. The layout adapts for both desktop and mobile devices.

## Features
- Responsive layout with mobile-first design.
- Three main sections:
  1. Details
  2. Contact (with animated banner and badge)
  3. Claim Form (with form validation UI)
- Custom styled components using plain CSS.
- Mobile view stacks sections in order: Details → Contact → ClaimForm.
- Form includes all essential input fields and styled submit buttons.

## Tech Stack
- React.js
- HTML, CSS (custom styling)
- React Icons (for icons)

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`

## Notes
- No backend functionality; this is a front-end UI clone.
- Built to match a provided UI layout.
